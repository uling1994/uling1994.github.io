2017年有事要做。

发表于 2017-02-07


首先，我要面对的是我曾经荒废的三年，真的想起来挺傻的，很盲目，浑浑噩噩的荒废了这么重要的三年，也深深地知道了人与人之间的差距到底是如何拉开的，自己以后不会再这么傻了…就这么健身练了三年的腱子肉，2333。

新的一年有新的计划:

  1. 把自己的博客要重写一遍了，不用别人的现成框架了，生命在于折腾。
  2. 把Vue学好，深入学好一个UI框架，react全家桶等以后技术栈需要的时候在进行考虑。
  3. Node.js 和 MongoDB的搭配怎么也得来一套。
  4. 深入Js，深入Js, 深入Js。其实我感觉css也挺有意思滴！

目标（送命题）

  1. TimeLine 项目必须在六月份之前上线，不能如期上线，就等着瘫痪吧……其实很简单的一个东西，还是自己太菜了。
  2. 自己在思索一下如何把高德地图公开的API和three.js做出3d的地图模型，大概是要向大漠老师的手淘团队取经了……毕竟双十一的天猫AR做的很有一套。
  3. 最后如期毕业，找到一份工作，比如搬砖，笑呵呵。

最后，皮皮虾我们走!
